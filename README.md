# Crypto RSI Alert Bot

Telegram bot that monitors RSI for crypto pairs and sends alerts when the market is overbought or oversold.

## Features
- RSI calculation
- Telegram notifications
- Anti-spam logic
- Configurable parameters

## Stack
- Python
- requests
- Binance API
- Telegram Bot API

## How to run
1. Create `.env` with BOT_TOKEN and CHAT_ID
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python main.py
